AmqpNotifier is a helper library to wrap Qusion with the AMQP gem

== Use
=== Publish
<pre><code>
AmqpNotifier.new('exchange', 'queue').publish('routing_key', 'message')
</code></pre>

=== Subscribe
<pre><code>
a = AmqpNotifier.new('exchange', 'queue').subscribe('routing_key') do |info, message|
  puts message
end
</code></pre>
